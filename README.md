# 🐕 Kostyan Mod — Minecraft 1.21.1 (Forge)

**Костян** — твой личный виртуальный помощник в Minecraft. Бессмертная прирученная собака с огромными знаниями об игре, которая защищает тебя и готова поговорить о чём угодно!

---

## ✨ Возможности

- 🐾 **Автоматический спаун** — Костян появляется рядом с тобой при входе в игру
- 💬 **Знания о Майнкрафте** — спрашивай про мобов, крафт, биомы, ресурсы и многое другое
- ⚔️ **Защита игрока** — атакует монстров одним ударом (100 урона!)
- ♾️ **Бессмертие** — Костяна невозможно убить, он всегда с тобой
- 📍 **Следование за игроком** — всегда рядом, телепортируется если потерялся
- 🇷🇺 **Поддержка русского языка** — интерфейс на русском!

---

## 💬 Использование

Чтобы поговорить с Костяном, используй команду в чате:

```
/kostyan <вопрос>
```

### Примеры вопросов:

| Команда | Ответ Костяна |
|---------|---------------|
| `/kostyan где найти алмазы` | Расскажет про уровень Y -60 |
| `/kostyan как убить крипера` | Объяснит тактику |
| `/kostyan нижний мир` | Расскажет про Нether |
| `/kostyan зачарование` | Объяснит систему зачарований |
| `/kostyan редстоун` | Введение в редстоун |
| `/kostyan привет` | Поприветствует тебя |

### Темы, о которых знает Костян:
- Мобы: крипер, эндермен, зомби, скелет, паук, дракон, виверн
- Ресурсы: алмазы, незерит
- Механики: крафт, зачарования, редстоун
- Измерения: нижний мир, край
- Выживание: еда, дом, ферма, строительство

## 📦 Установка

### Требования:
- Minecraft **1.21.1**
- **Minecraft Forge 51.0.33+**
- Java 21+

### Шаги:
1. Скачай [Forge 1.21.1](https://files.minecraftforge.net/net/minecraftforge/forge/index_1.21.1.html) и установи его
2. Скачай файл `kostyanmod-1.0.0.jar` из [Releases](../../releases)
3. Положи `.jar` в папку `mods` в директории Minecraft
4. Запускай игру через профиль Forge
5. Заходи в мир — Костян сразу появится рядом!

## 🔨 Сборка из исходников

```bash
# Клонируй репозиторий
git clone https://github.com/ВАШ_ЛОГИН/kostyan-mod.git
cd kostyan-mod

# Собери мод
./gradlew build

# JAR файл будет в build/libs/
```

### Запуск в тестовой среде:
```bash
./gradlew runClient   # Запустить клиент Minecraft с модом
./gradlew runServer   # Запустить сервер с модом
```

## 🎨 Добавление текстуры

В папку `src/main/resources/assets/kostyanmod/textures/entity/` нужно добавить файл `kostyan.png` (текстура 64x32 пикселей, как у обычной собаки в Minecraft).

Ты можешь:
1. Скопировать текстуру волка из ресурсов Minecraft и перекрасить
2. Нарисовать свою в [Blockbench](https://www.blockbench.net/)
